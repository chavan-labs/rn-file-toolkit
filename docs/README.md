# rn-file-toolkit documentation

A native file management toolkit for React Native and Expo custom dev clients.

## What you get

- Background-capable downloads
- Multipart uploads
- Filesystem API
- Zip/unzip utilities
- Base64 and media helpers

## Install

```bash
npm install rn-file-toolkit
```

Then run iOS pods in your app project if needed.
